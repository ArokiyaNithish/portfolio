VisionSafe: AI-Powered Traffic Collision Prevention System with ESP32-CAM, Deep Learning & App Integration


Problem Statement :

Traditional traffic systems lack intelligent accident detection and real-time alerts, causing delays in emergency response and increased fatalities.

Tools & Technologies :

ESP32-CAM, YOLOv4-Tiny (Object Detection), Deep Learning (TensorFlow Lite), IoT Cloud Dashboard, Custom Android App, Python, OpenCV


Technical Explanation :

The project uses ESP32-CAM for real-time video surveillance at traffic zones. YOLOv4-Tiny model is trained to detect accidents, fallen bikes, and collisions. Once an accident is detected, the ESP32 triggers an alert through an IoT app that notifies nearby emergency services and displays the location using GPS integration. It also logs events in a cloud database for analysis. The custom app provides a user interface for monitoring and control.


Your Role :

  • Trained YOLOv4-Tiny on traffic accident datasets.
  • Deployed the model onto ESP32-CAM for real-time inference.
  • Built and linked the mobile app for emergency alerts.

Key Features:

  • Real-time accident detection.
  • Emergency alerts with GPS.
  • Cloud logging and mobile access.

Outcome or Impact :

Improves emergency response times and reduces fatalities in road accidents. Enables authorities to monitor high-risk areas using AI and IoT.

AgriBot automates the farming process using Arduino Uno as the brain. Soil moisture sensors detect irrigation needs and control the water pump via a relay. SmartCart: Arduino-Based Intelligent Shopping Trolley with Automated Billing System


Problem Statement :

Manual shopping carts require customers to wait in long queues for billing, resulting in inefficiencies and poor shopping experience. The lack of automation leads to errors and time wastage at checkout counters.

Tools & Technologies :

Arduino Uno, RFID Reader, IR Sensors, Load Cell, LCD Display, Embedded C with Arduino IDE


Technical Explanation :

The SmartCart system is built on Arduino Uno, integrating RFID readers, IR sensors, and a load cell to automate in-cart billing. When a product with an RFID tag is placed in the trolley, the tag is read, and the item data is processed. IR sensors verify if the item remains in the trolley, and the load cell confirms the weight. All this data is displayed on an LCD screen in real-time. The final bill is generated instantly, allowing users to skip manual billing queues. This prototype shows how microcontrollers and sensors can automate and digitize retail shopping experiences.


Your Role :

  • Designed the hardware layout and wiring of sensors.
  • Programmed the Arduino Uno for item detection and real-time billing display.
  • Tested and optimized sensor placement for accurate tracking.

Key Features:

  • Real-time item tracking and billing.
  • Eliminates checkout queue.
  • Easy-to-use display interface.

Outcome or Impact :

The project reduces billing time, enhances customer experience, and can be deployed in smart retail environments. It demonstrates the potential of embedded systems to solve real-world retail problems.
